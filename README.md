React ToDos List
A simple, responsive, and functional Task Management application built using React.js. This app allows users to add new tasks, delete completed ones, and navigate through different sections using a single-page application (SPA) approach.
Features: 
1) Add ToDos: Add task with title and description.
2) Delete Tasks: Remove task once they are finish.
3) Responsive Design: Styled with Bootstrap 5 for a clean look on all devices.
4) Routing: Seamless navigation between Home and About pages using `react-router-dom`.

Tech stack: React.js and Bootstrap.
Installation and setup:
Clone the repository
Navigate into the folder: cd todos-list
Install dependencies: npm install
Run the application: npm start
